This project is a Flask-based web application that:
1. Scrapes trending topics from Twitter using a Selenium script.
2. Stores the scraped data in a MongoDB database.
3. Displays both the latest data (from the script) and all stored records (from MongoDB) on a web page.

---

## **Features**

- **Dynamic Data Fetching**: Fetches the latest trending topics from Twitter.
- **MongoDB Integration**: Stores and displays all previously scraped records.
- **User Interface**: Simple web interface to trigger data scraping and view results.

---

## **Getting Started**

### **Prerequisites**

1. **Python**: Version 3.7 or higher.
2. **MongoDB**: Install and run MongoDB on your system. [Installation Guide](https://www.mongodb.com/docs/manual/installation/)
3. **Google Chrome**: Ensure Google Chrome is installed for Selenium.
4. **ChromeDriver**: Install ChromeDriver compatible with your Chrome version. [Download ChromeDriver](https://chromedriver.chromium.org/downloads)

### **Libraries**

Install required Python libraries:
```bash
pip install flask selenium beautifulsoup4 pymongo requests

## **Usage**

### **Run Script**

* Click the "Run Script" button to fetch the latest trending topics from Twitter.

### **View Data**

* **Top Section** : Displays the latest data fetched from Twitter.
* **Bottom Section** : Displays all previously scraped records stored in MongoDB.

---

## **Code Explanation**

### **Flask App (`app.py`)**

1. **Routes** :

* `/`: Displays the web interface with fresh data and all MongoDB records.
* `/run-script`: Runs the Selenium script to scrape data from Twitter.

1. **Functions** :

* `fetch_trending_topics`: Scrapes Twitter trending topics using Selenium and stores them in MongoDB.

### **Web Page (`index.html`)**

* **Run Script Button** :
* Fetches data from the `/run-script` endpoint and displays it dynamically.
* **All Records Section** :
* Displays all records fetched from MongoDB.

## **Dependencies**

1. **Flask** : For building the web application.
2. **Selenium** : For web scraping.
3. **BeautifulSoup** : For parsing HTML content.
4. **pymongo** : For MongoDB integration.
5. **Requests** : For fetching IP address details.

## **Notes**
